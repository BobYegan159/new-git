գտակար Կայքեր

Այս կայքը նվիրված է ծրագրավորման տարբեր լեզուների և տեխնոլոգիաների, համակարգիչների, բջջային հարթակների և ՏՏ տեխնոլոգիաների:

Կայքում կարող եք գտնել նյութեր այնպիսի ոլորտներում, ինչպիսիք են C# լեզուն և .NET տեխնոլոգիաների ընտանիքը (ASP.NET, WPF, Entity Framework և այլն), Java-ի վրա հիմնված տեխնոլոգիաները (Java SE, Android ծրագրավորում և այլն), Python, Dart (ներառյալ Flutter), աշխատել տվյալների բազաների հետ (MS SQL Server, MySQL, MongoDB), ինչպես նաև WEB տեխնոլոգիաների, ինչպիսիք են HTML5, AJAX, jQuery, Node.js, ExtJS, Angular, React և այլն:

Ինտերնետում վեբ ծրագրավորողների ամենամեծ կայքը: Ամեն տարի ցուցադրվում է 3 միլիարդ էջ: Ամեն ամիս 60 միլիոն այցելու: Ուսուցման վեբ մշակումը պետք է լինի հեշտ հասկանալի և հասանելի բոլորի համար, ամենուր:

W3Schools-ը ստեղծվել է 1998 թվականին և իր անվանումը ստացել է Համաշխարհային ցանցից (WWW), սակայն փոխկապակցված չէ W3C-ի հետ:

Բարի գալուստ MDN ուսումնական տարածք: Հոդվածների այս հավաքածուն նպատակ ունի ուղղորդել ամբողջական սկսնակներին դեպի վեբ մշակում այն ​​ամենով, ինչ նրանց անհրաժեշտ է կայքերը կոդավորելու համար:

MDN-ի այս տարածքի նպատակն է ոչ թե ձեզ «սկսնակից» հասցնել «փորձագետի», այլ ձեզ «սկսնակից» դեպի «հարմարավետ»: Այնտեղից դուք պետք է կարողանաք սկսել ձեր ճանապարհը, սովորելով մնացած MDN-ից և այլ միջանկյալից մինչև առաջադեմ ռեսուրսներից, որոնք ենթադրում են շատ նախկին գիտելիքներ:

ՏՏ Ոլորտի մարդիկ

Թիմոթի Ջոն Բերներս-Լի բրիտանացի գիտնական, URI, URL, HTTP, HTML-ի գյուտարար, Համաշխարհային ցանցի ստեղծող (Ռոբերտ Քայոյի հետ) Համաշխարհային ցանցի կոնսորցիումի գործադիր տնօրեն։

Ջեյմս Արթուր Գոսլինգ կանադացի համակարգչային մասնագետ. Հայտնի է որպես Java ծրագրավորման լեզվի հիմնադիր։

Բրենդան Այք - մշակող JavaScript ծրագրավորման լեզվի հեղինակ: Նա համարվում է «Mozilla» ընկերության հիմնադիրներից մեկը։

Լոուրենս Էդվարդ Փեյջը ամերիկացի համակարգչային գիտնական և ինտերնետ-ձեռներեց է: Նա առավել հայտնի է որպես Google-ի համահիմնադիրներից մեկը։
